# 🌸 Period Tracker

A charming, aesthetic, and privacy-focused wellness journal built to help individuals track their menstrual cycles and understand their bodies better.

🎨 **Live Link:** [View My Live Deployment Hosted on Vercel](https://period-tracker-murex-five.vercel.app/)

---

## ✨ Features

* **Charming Aesthetic Layout:** Designed with a cozy, professional Chocolate Brown & Warm Beige palette featuring traditional serif typography.
* **Custom User Onboarding:** A tailored, secure introductory form that captures your username, average cycle length, and previous period dates without tracking your data remotely.
* **Intelligent Prediction Engine:** Automatically calculates and forecasts future period start windows, highlighting them with a soft, reddish-peach clay tint.
* **Overriding Native Components:** Replaced generic browser pop-up calendars with bespoke, custom-themed dropdown selection arrays to match the theme perfectly.
* **Dynamic Holistic Care Tips:** Interactive symptom trackers (Cramps, Headaches, Bloating, Fatigue, Mood) that instantly provide supportive wellness tips and advice when clicked.
* **Local Storage Persistence:** Uses native browser cache storage so your private profile and logs are saved securely right on your own device.

---

## 🛠️ Technology Stack

* **Frontend Library:** React (Functional Components & Hooks)
* **Build Tooling:** Vite
* **Styling:** Custom Vanilla CSS3
* **Components:** React-Calendar Layout Overrides
* **Deployment:** Vercel

## 🚀 How to Run Locally

If you want to clone this project and run it on your local system, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/YOUR_GITHUB_USERNAME/menstrual-cycle-tracker.git](https://github.com/YOUR_GITHUB_USERNAME/menstrual-cycle-tracker.git)
    ```
2.  **Navigate into the directory:**
    ```bash
    cd menstrual-cycle-tracker
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Launch the development server:**
    ```bash
    npm run dev
    ```
    Open `http://localhost:5173` in your browser to view the application!
